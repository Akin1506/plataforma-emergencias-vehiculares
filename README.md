# Plataforma de Emergencias Vehiculares

Proyecto académico para la gestión de emergencias vehiculares.

## Funcionalidades

- Registro de incidentes
- Visualización de incidentes
- Asignación de talleres
- Actualización del estado del incidente

## Tecnologías utilizadas

### Backend
- FastAPI
- Python

### Frontend
- HTML
- CSS
- JavaScript

## Estructura del proyecto

- `backend/`: API y lógica del sistema
- `frontend/`: interfaz web básica
